-- A Tiny proof of concept of using knockout with redux --

README 

To get started,

1. Open the folder in the console
2. Type in 'npm install'
3. After downloading the modules, type in 'gulp start'
4. Open browser and type in localhost address as seen in terminal:

	[Browsersync] Access URLs:
	 ---------------------------------------
	       Local: http://localhost:3000
	 ---------------------------------------
	          UI: http://localhost:3001
